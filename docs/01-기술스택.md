# 전자정부프레임워크 CMS 기술 문서

## 프로젝트 개요

전자정부 표준프레임워크 v4.3 기반 CMS(Content Management System) 개발 프로젝트

---

## 버전 정보

| 구분 | 버전 | 출시일 |
|------|------|--------|
| **전자정부프레임워크** | 4.3.0 | 2024.12 |
| **공통컴포넌트** | 4.3.1 | 2025.04.29 |
| **Spring Framework** | 5.3.37 | - |
| **Spring Boot** | 2.x | - |
| **Java** | 11+ | - |

---

## 기술 스택

### Backend

| 기술 | 선택 | 이유 |
|------|------|------|
| **빌드 도구** | Maven | 전자정부 공식 예제 기준 |
| **Framework** | Spring Boot 2.x | 전자정부프레임워크 4.3 기반 |
| **ORM** | MyBatis | 전자정부 기본 설정 |
| **Database** | MySQL 8.0 | 국내 레퍼런스 풍부 |

### Frontend

| 기술 | 선택 | 비고 |
|------|------|------|
| **Framework** | React | egovframe-template-simple-react 활용 |
| **UI** | KRDS (Korea Design System) | 정부 표준 디자인 시스템 |

## 개발 환경

### 필수 설치

```
- JDK 11+
- Maven 3.8+
- MySQL 8.0+
- Node.js 18+ (React 프론트엔드)
- IDE: IntelliJ IDEA 또는 Eclipse (전자정부 개발환경 4.3)
```

### 지원 데이터베이스

전자정부프레임워크 공통컴포넌트는 8종 DB를 지원:
- MySQL
- MariaDB
- PostgreSQL
- Oracle
- Tibero
- Altibase
- CUBRID
- Goldilocks
