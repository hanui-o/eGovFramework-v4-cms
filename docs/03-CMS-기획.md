# CMS 기획서

## 프로젝트 개요

| 항목 | 내용 |
|------|------|
| **유형** | 기업/기관 홈페이지 CMS |
| **사용자** | 관리자 + 일반 회원 |
| **프레임워크** | 전자정부프레임워크 v4.3 |
| **목표** | 1차 MVP 개발 후 점진적 확장 |

---

## 사용자 역할

| 역할 | 설명 | 주요 권한 |
|------|------|----------|
| **슈퍼관리자** | 시스템 전체 관리 | 모든 권한 |
| **관리자** | 콘텐츠 관리 | 게시판, 회원, 통계 관리 |
| **일반회원** | 서비스 이용 | 글 작성, 댓글, 문의 |
| **비회원** | 열람만 가능 | 공개 게시판 조회 |

## 화면 구성

### 사용자 화면 (Frontend)

```
메인 페이지
├── 헤더 (로고, 메뉴, 로그인/회원가입)
├── 메인 비주얼 (슬라이드 배너)
├── 공지사항 미리보기
├── 주요 서비스 바로가기
├── 팝업 레이어
└── 푸터 (기관 정보, 링크)

서브 페이지
├── 기관 소개
├── 게시판 (공지/자유/자료실/갤러리)
├── FAQ / Q&A
├── 마이페이지
└── 회원가입 / 로그인
```

### 관리자 화면 (Admin)

```
관리자 대시보드
├── 오늘의 통계 (접속자, 신규회원, 새 글)
├── 최근 게시물
├── 최근 Q&A
└── 시스템 상태

관리 메뉴
├── 회원 관리
├── 게시판 관리
├── 콘텐츠 관리 (팝업, 배너)
├── 고객지원 (FAQ, Q&A)
├── 통계
└── 시스템 설정
```

## 개발 우선순위

### Phase 1: 핵심 기능 (필수)

```
1. 회원가입 / 로그인 / 로그아웃
2. 권한 관리 (관리자/회원 구분)
3. 게시판 CRUD (공지사항)
4. 관리자 대시보드 기본
```

### Phase 2: 확장 기능

```
5. 게시판 확장 (자유게시판, 자료실, 갤러리)
6. 댓글 기능
7. FAQ / Q&A
8. 마이페이지
```

### Phase 3: 부가 기능

```
9. 팝업 / 배너 관리
10. 통계 (접속, 게시물, 회원)
11. 메뉴 관리
12. 약관 / 개인정보처리방침
```
